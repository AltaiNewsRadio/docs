# Altai News Radio API

Welcome to the **Altai News Radio (ANR) API**.

This API provides programmatic access to:

- System readiness and governance signals
- Promotion eligibility logic
- Deep geopolitical assessments
- Admin-only ingestion and control endpoints

---

## Base URL

https://us-central1-anrtestmap.cloudfunctions.net

---

## Authentication

All protected endpoints require an API key:

x-api-key: aie_live_XXXXXXXX

Admin-only endpoints require:

x-admin-key: \<admin-secret\>

## Design principles

- Immutable records
- Explicit governance gates
- No silent learning or optimization
- Transparency over performance hacks
